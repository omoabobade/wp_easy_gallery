== Description ==

This is an easy to use gallery plugin that makes use of the already existing WordPress Media Uploader. The gallery will incorporate several popular online Jquery gallery library script such as gallerific, desoslide etc. User will get the option of choosing the gallery library they want to use for respective gallery.


== Installation ==

1. Download the plugin 
2. Install plugin by Uploading it into the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Create a new gallery by clicking on the menu by the left side of the admin menu
5. Copy the shortcode of the respective gallery into where you want it to show
